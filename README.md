# Build With Envi

1. PHP 8.2.12
2. MariaDB 10.1.28
3. Node v20.19.0

# How It Works

1. RUN Command
    - php artisan migrate
    - RUN php artisan cache:clear
    - RUN php artisan config:clear
    - RUN php artisan config:cache
    - RUN php artisan storage:link
2. RUN npm install && npm run build
3. composer run dev

    (Run on port 8000)
